# rtmp-ocx-H265-
rtmp 直播控件(ocx)，支持H265,支持最多十六个窗口，提供测试代码
Bin目录中有可以执行文件testmyctrl.exe，运行前需要手工注册bin目录中的blvideo.ocx,cmd命令: regsvr32 你的目录\bin\blvideo.ocx
testmyctrl目录是调用控件的代码
